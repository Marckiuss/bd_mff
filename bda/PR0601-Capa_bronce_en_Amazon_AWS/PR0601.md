# PR0601. Capa bronce en Amazon AWS

## 1. Contexto
La Secretaría de Estado de Turismo quiere desarrollar un sistema de inteligencia de datos para optimizar la gestión de las playas españolas. Para ello, necesitan centralizar en un Data Lake en AWS (Capa Bronce) dos fuentes de información críticas:

**Catálogo Nacional de Playas**: un dataset detallado con servicios, accesibilidad y características físicas.

**Predicción Meteorológica**: datos en tiempo real para las zonas costeras (fuente AEMET).

Tu misión es automatizar la ingesta de estos datos hacia Amazon S3 utilizando Python.

## 2. Objetivos de la práctica
**Consumo de API**: gestionar la autenticación y descarga de archivos JSON desde AEMET OpenData.

**Gestión de datos maestros**: procesar un dataset complejo en formato CSV sobre la infraestructura de las playas.

**Arquitectura Cloud**: implementar una estructura de almacenamiento profesional y escalable en S3.

**Seguridad**: administrar secretos y claves de acceso mediante variables de entorno.

## 3. Fuentes de datos

#### A. Catálogo de playas (CSV)
Se proporciona un archivo con información exhaustiva de las playas. También lo puedes descargar del Portal de Datos Abiertos de ESRI.

Los campos clave para futuras analíticas son:

**Nombre, Provincia y Término_M (Municipio).**

**Servicios**: Duchas, Aseos, Acceso_dis, Bandera_az.

**Ubicación**: Latitud, Longitud y las coordenadas X, Y.

**Estado**: Grado_ocup y Grado_urba.

#### B. API AEMET OpenData (API REST)
**Servicio**: predicción para playas (específicamente el endpoint de predicción diaria).

**Requisito**: obtener el JSON de predicción para las provincias presentes en el CSV.

**Nota técnica**: el alumno deberá manejar el flujo de la API (la API de AEMET devuelve un JSON con una URL, de la cual se descarga el dato final).

## 4. Requerimientos técnicos
### Fase 1: Estructura de la Capa Bronce en S3
Debes crear un bucket y organizar los datos siguiendo un esquema de particionamiento y versionado:

**bronce/catalogos/guia_playas/v1/playas.csv**

**bronce/meteorologia/prediccion_playas/year=YYYY/month=MM/day=DD/**

### Fase 2: Desarrollo del ingestor (Python)

Desarrolla un script modular que realice lo siguiente:

**Módulo de carga local**: leer el CSV de playas y subirlo a S3 asegurando que el nombre del archivo incluya el origen de los datos.

**Módulo API AEMET**: conectar con la API, extraer la predicción del día y guardarla directamente en S3 como un archivo JSON.

**Trazabilidad**: cada archivo subido debe contener metadatos o un nombre que permita saber en qué fecha exacta se realizó la ingesta.

```python
    }

print("Iniciando la subida a la Capa Bronce...")

# Iteramos el dicionario
for archivo in archivos_a_subir:

    ruta_s3 = archivo["s3"]
    
    print(f"\nProcesando: {ruta_local} ...")
    
    # Comprobamos que el archivo existe localmente antes de intentar subirlo
    if os.path.exists(ruta_local):
        try:
            # 4. Ejecutamos la subida a S3

        except Exception as e:
            print(f"Error al intentar subir '{ruta_local}' a S3.")
            print("Detalle:", e)
    else:
        print(f"Error: No encuentro el archivo en la ruta local '{ruta_local}'.")

print("\nProceso de subida finalizado.")
```

```python
import requests
import pandas as pd

ahora = datetime.now()
year = ahora.strftime("%Y")
month = ahora.strftime("%m")
day = ahora.strftime("%d")

carpeta_s3 = f"bronce/meteorologia/prediccion_playas/year={year}/month={month}/day={day}/"
print(f"Carpeta destino: {carpeta_s3}")

headers_aemet = {
    'api_key': AEMET_API_KEY,
    'Accept': 'application/json'
}

for index, fila in df_prueba.iterrows():
    codigo = fila['ID_PLAYA'] 
    nombre_limpio = str(fila['Nombre']).replace(" ", "_").replace("/", "-")
    
    print(f"\nProcesando: {nombre_limpio} (Código: {codigo})")
    
    url_peticion = f"https://opendata.aemet.es/opendata/api/prediccion/especifica/playa/{codigo}"
    
    try:
        respuesta_1 = requests.get(url_peticion, headers=headers_aemet)
        datos_json = respuesta_1.json()
        
        if respuesta_1.status_code == 200 and 'datos' in datos_json:
            url_descarga_final = datos_json['datos']
            
            respuesta_2 = requests.get(url_descarga_final)
            prediccion_texto = respuesta_2.text 
            
            nombre_archivo_s3 = f"{carpeta_s3}prediccion_{codigo}_{nombre_limpio}.json"
            
            s3_client.put_object(
                Bucket=BUCKET_NAME,
                Key=nombre_archivo_s3,
                Body=prediccion_texto,
                ContentType='application/json' 
            )
            print(f"  ✅ Guardado en S3: {nombre_archivo_s3}")
            
        else:
            print(f"Error AEMET: {datos_json.get('descripcion', 'Error desconocido')}")
            
        time.sleep(1) 
        
    except Exception as e:
        print(f"Fallo técnico al procesar {nombre_limpio}: {e}")

print("\nMódulo de Ingesta AEMET completado con éxito.")
```
